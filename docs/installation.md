# Installation

## Requirements

- Python 3.9 or higher
- pip or uv package manager

## Install from PyPI

```bash
pip install pflow-cli
```

## Install from Source

```bash
git clone https://github.com/spinje/pflow.git
cd pflow
make install
```

## Verify Installation

```bash
pflow --version
```

## Quick Start

Once installed, you can start using pflow immediately:

```bash
# Run a simple workflow
pflow ...

# Use pipe syntax
pflow ...
```
